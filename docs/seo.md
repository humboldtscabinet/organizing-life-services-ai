# SEO Integrations

## Phase 1 — Data Sources
- Google Search Console (GSC)
- Google Analytics 4 (GA4)
- Google Business Profile (GBP)
- Google Ads

## Workflow
1. Pull data from Google APIs → store in Postgres
2. Run audits / generate recommendations
3. Push results to Google Sheets for manual review
4. Human approves or rejects each action

Full implementation details added as each integration is built.
